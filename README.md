# SOD Club Frontend for Round 1

## Project Structure

```
sod-quiz-platform/
├── frontend/          ← React app (Vite)
│   ├── src/
│   │   └── App.jsx    ← Main application
│   ├── index.html

## Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
# Runs on http://localhost:3000 locally
```
